# artha-js

artha-js is a lightweight, zero-dependency JavaScript library that provides a set of custom HTML elements (Web Components) and core utilities for building dynamic, data-driven single-page applications. It is designed to work directly in the browser with native ESM imports — no build tools or bundler required.

## Key features

  </Card>

## How it works

artha-js follows an HTML-first philosophy. You declare your UI structure in HTML, add a few attributes, and the library handles data fetching, rendering, and feedback automatically.

```html index.html
<!DOCTYPE html>
<html lang="en">
<head>
  <link rel="stylesheet" href="dist/artha.min.css">
  <script type="module" src="dist/artha.min.js"></script>
</head>
<body>
  <!-- Fetch /api/users and render each item using the template -->
  <artha-container action="/api/users" template="user-template">
    <artha-message></artha-message>
  </artha-container>

  <template id="user-template">
    <div class="user-card">
      <strong data-wire="name"></strong>
      <span data-wire="email"></span>
    </div>
  </template>
</body>
</html>
```

The `data-wire` attribute maps JSON keys from the server response directly to DOM elements — no JavaScript required.

## Project structure

```
artha-js/
├── src/
│   ├── core/
│   │   ├── Config.js       # Global configuration
│   │   ├── XHR.js          # XMLHttpRequest wrapper
│   │   ├── EventBus.js     # Pub/sub event system
│   │   ├── TaskQueue.js    # Async task tracking
│   │   └── Util.js         # DOM helpers
│   └── components/
│       ├── artha-container.js   # Dynamic data container
│       ├── artha-form.js        # XHR-powered form
│       └── artha-message.js     # Status message display
├── dist/
│   ├── artha.min.js        # Bundled output
│   └── artha.min.css       # Compiled styles
└── index.html              # Demo page
```

## Browser compatibility

artha-js uses modern browser APIs including Custom Elements v1, `XMLHttpRequest`, `EventTarget`, `FormData`, and ES modules. It targets evergreen browsers (Chrome, Firefox, Safari, Edge).

<Note>
  artha-js is designed to be included as a compiled bundle (`dist/artha.min.js`). You can also import individual modules directly via ESM if you use a bundler.
</Note>
